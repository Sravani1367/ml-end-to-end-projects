# 🤖 ML End-to-End Projects

A collection of end-to-end machine learning projects covering data preprocessing, model building, evaluation, and insights — built to solve real-world problems.

---

## 📁 Projects

### 🏠 House Price Prediction
> Predicting median house prices based on various features like location, size, and demographics.

- Performed exploratory data analysis (EDA) on housing data
- Cleaned and preprocessed data using Pandas & NumPy
- Built and evaluated regression models using Scikit-learn
- Visualized trends and feature correlations using Matplotlib & Seaborn

**Dataset:** `housing.csv`
**Notebook:** `median_house.ipynb`

---

### 🏥 Disease Prediction
> Predicting the likelihood of disease based on patient health parameters.

- Analyzed medical dataset to identify key health indicators
- Preprocessed and engineered features for better model performance
- Trained classification models using Scikit-learn
- Evaluated model accuracy, precision, and recall

## 🚀 How to Run

```bash

# Navigate into a project folder

# Install dependencies
pip install pandas numpy scikit-learn matplotlib seaborn jupyter

# Launch Jupyter Notebook
jupyter notebook
```
